# 打印设置指南（Windows + 针式打印机 + 14×22cm 连续三联纸）

本文档面向 IT 与现场仓管，描述工厂端如何把 CloudPivot 的自由出入库打印功能配置到位。**业务上线前必读**。

## 前置条件

- CloudPivot 桌面版（含本次打印功能） 已经安装到工位电脑
- 针式打印机型号示例：Epson LQ-630K / LQ-690K（或同类 9 / 24 针机型）
- 连续三联纸规格：**14cm × 22cm**（5.5"×8.66"），白/红/黄三联预印刷

## 步骤一：安装打印机驱动

按打印机厂商说明安装驱动。一般 Windows 自带 Epson LQ 系列驱动；若没有，从厂商官网下载。

安装完后在「设置 → 设备 → 打印机和扫描仪」里看到设备就 OK。

## 步骤二：定义自定义纸张尺寸

这是**最关键的一步**，必须做，否则 CloudPivot 打印对话框出来不会有 14×22cm 选项。

1. 打开「控制面板 → 设备和打印机」（Win 10 / 11 都能搜到这个旧界面）
2. 在窗口顶部菜单点「打印服务器属性」（Print Server Properties）
3. 切换到「**纸张**」（Forms）选项卡
4. 勾选「创建新纸张」（Create a new form）
5. 表单名称填：`三联纸 140x220`（你能记住的中文名）
6. 纸张大小：
   - 单位选「公制」
   - 宽度：`14.00cm`
   - 高度：`22.00cm`
7. 边距：上下左右都填 `0.50cm`（与 CloudPivot 默认边距对齐）
8. 点「保存表单」「确定」

## 步骤三：把这张自定义纸设为打印机默认

1. 回到「设备和打印机」
2. 右键针式打印机 →「打印首选项」
3. 在「纸张/质量」选项卡里把「纸张尺寸」改成刚才创建的「三联纸 140x220」
4. 点「应用」「确定」
5. 再右键打印机 →「打印机属性」→「高级」选项卡
6. 默认值修改为同一种纸张

## 步骤四：在 CloudPivot 里走一次打印

1. 用任意账号登录 CloudPivot
2. 进「自由出入库」模块
3. 找一张状态为「已过账」的单据
4. 在操作列点「**打印单据**」图标
5. 浏览器打印对话框出现：
   - 目标打印机：选刚才配置的针式打印机
   - 纸张尺寸：应自动选中「三联纸 140x220」或「自定义」
   - 颜色：黑白
   - 缩放：100%（**不要勾选「适合页面」或「缩小至适合纸张」**，否则字符会缩小错位）
   - 页边距：默认 / 无 都行
6. 点「打印」

## 验收 3 条硬指标

- 纸张被识别为 14×22cm 自定义 ✅
- 中文宋体 + 越南文 Noto 字符清晰可辨，数字不错位 ✅
- 上/下/左/右 4 个边距与三联纸预印框 ± 2mm ✅

三条全过 → OK 上线。
任一不过 → 联系开发者，可能需要切换到后端 PDF 生成方案（约 1-1.5 周）。

## 常见问题

**Q: 出纸位置偏 1cm 左右，怎么调？**
A: 在「打印首选项 → 高级」里调「页边距」或在打印机厂商驱动里调"水平偏移 / 垂直偏移"。先调 ± 2mm 看效果。

**Q: 中文显示成方块？**
A: 打印机驱动里把 "字体替换" 关掉，或在「设备和打印机 → 属性 → 字体」选项卡里选「下载为软字体」。

**Q: 续页表头不打印？**
A: 这是 CloudPivot 的 CSS 设置（`thead { display: table-header-group }`），如果不生效请联系开发者。

**Q: 三联纸只打第一联，二/三联是空白？**
A: 这是连续纸**机械复写**，三联是靠针击打传力的。如果二/三联空白，说明：① 三联纸已过期复写层老化，换新纸；② 打印浓度太低，在打印机自身菜单里调高浓度；③ 三联纸顺序装反了。
