# 打印模板设计器 — 业务快速入门

本文档面向**业务负责人 / 仓管主管**，介绍如何在 CloudPivot 里调整自由出入库（以及其他单据）的打印样式。

## 设计器在哪里

1. 登录 CloudPivot（admin 角色）
2. 进「系统设置 → 打印设置」
3. 在「固定模板选择」卡片右上角点「**编辑模板**」按钮

设计器有三栏：

```
┌─────────────┬─────────────────────┬─────────────┐
│  可用字段    │   列顺序 + 预览       │  列属性      │
│  (左)        │  (中)                │  (右)        │
└─────────────┴─────────────────────┴─────────────┘
```

## 能改什么（v1）

### ① 显示哪些列
- 在「列顺序」面板里，每行左边的复选框控制该列是否打印
- 复选框去掉 → 这列不出现在打印纸上
- 复选框勾上 → 这列出现

### ② 列的顺序
- 选中某列，点它行末的 ⬆ ⬇ 按钮上移 / 下移
- 顺序即"打印纸上从左到右"的顺序

### ③ 加一列 / 减一列
- 加：左侧「可用字段」点字段右边的 ➕，该字段添加到列表末尾
- 减：右侧选中后，行末点 🗑️ 删掉

### ④ 列宽
- 中间面板点中某列 → 右侧属性面板出现
- 「列宽」下拉有 6 档：4 / 6 / 8 / 10 / 12 / 16 字符宽
- 选一档后，中间预览实时刷新

### ⑤ 对齐
- 右侧属性面板「对齐」三档：左 / 中 / 右
- 一般文字选左、数字选右、状态选中

## 不能改什么（v1）

这些功能在 v2 里再做：

- ❌ 字号（v1 锁死 10pt）
- ❌ 列标题文案（v1 用默认中文标题）
- ❌ 页眉/页脚自定义
- ❌ 整页布局（v1 是固定的"页眉 + 表格 + 页脚"模板）

## 标准操作流程（SOP）

### 第一次调
1. 进设计器，预览看默认样式 OK 不 OK
2. 不 OK 就动一动列顺序 / 列宽 / 对齐
3. 实时预览看效果
4. 满意后点右上角「**保存**」
5. 回到打印设置页 → 这个模板会显示「已自定义」标签

### 改坏了想重置
1. 设计器右上角「**重置为默认**」
2. 弹出确认 → 确认
3. 整个模板配置回到出厂默认（你的所有调整丢失）

## 注意事项

- ⚠️ **保存即生效**：保存后下一次打印就用新样式。如果有人正在打印中，不影响他这次（他用的是开始打印那一刻的样式）
- ⚠️ **每个模板独立**：自由出入库 / 采购入库 / 销售出库等是不同的模板，互不影响
- ⚠️ **权限**：只有 admin 角色能进设计器和保存；operator 看不到「编辑模板」按钮

## 常见问题

**Q: 我加了一列但预览没显示？**
A: 检查右侧属性面板里"可见"复选框是不是勾上的，再检查"列宽"是不是 0（v1 最小 4 字符宽）。

**Q: 保存后打印的还是老样式？**
A: 关掉旧的打印对话框，重新点列表的打印按钮。如果还是老样式，刷新页面再试一次。

**Q: 设计器看不到某个字段（比如成本单价）？**
A: 该字段不在自由出入库的数据源里。v1 阶段如果需要新字段，联系开发者。

**Q: 想给打印加我的公司 Logo？**
A: 回到「打印设置」主页 → 「显示项」勾上「打印 Logo」+ 「打印公司信息」+ 保存。Logo 图片本身上传由 IT 配（v1 通过修改 `system_config` 数据库表的 `print_company_logo` 字段）。

**Q: 我能调字号吗？**
A: v1 不能。v2 计划开放 10pt / 12pt 两档下拉。如果现场反馈强烈，可以推进 v2。
