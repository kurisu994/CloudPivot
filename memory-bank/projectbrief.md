# Project Brief — 云枢 (CloudPivot IMS)

## 核心愿景

为越南中小型家具生产工厂打造一套 **轻量级桌面端进销存管理系统**，覆盖采购→生产→销售→库存→财务全链路，支持中/越/英三语与 VND/CNY/USD 三币种。

## 产品名称

- 中文名：**云枢**
- 英文名：**CloudPivot IMS** (Inventory Management System)
- 标识符：`com.cloudpivot.ims`

## 项目范围（v1.0 已完成）

### 12 大功能模块

1. **基础数据**：物料 / 分类（树形）/ 供应商 / 客户 / 仓库 / 单位
2. **BOM 管理**：多级物料清单 / 反查 / 需求计算 / 成本核算
3. **采购管理**：采购单 → 入库（批次 + 费用分摊）→ 退货（成本回调）
4. **销售管理**：销售单（行折扣 + 整单折扣率）→ 出库（FIFO + 双成本快照）→ 退货（成本回调 + 应收冲减）
5. **库存管理**：实时库存查询 / 出入库流水 / 自由出入库（草稿 + 原子过账）/ 盘点 / 调拨
6. **定制单管理**：配置明细 / 定制 BOM / 预留释放 / 报价 / 转销售 / 转生产
7. **生产工单**：领料出库 / 退料入库 / 完工入库（含成本核算）
8. **智能补货**：策略配置 / 建议计算 / 消耗趋势 / 一键生成采购单
9. **财务辅助**：应付账款 / 应收账款 / 付款收款登记
10. **报表中心**：采购 / 销售 / 库存（收发存 / 库龄 / 滞销 / 趋势）+ Excel 导出
11. **系统设置**：企业信息 / 显示偏好 / 编码规则 / 库存规则 / 汇率 / 打印 / 数据管理 / 操作日志 / 用户管理
12. **认证与权限**：多账号 / admin·operator·viewer 三角色 / 模块×操作二维权限矩阵

### v1.0 范围边界

- **是**：业务单据、库存、报表、打印和简易应收应付
- **不是**：专业财务软件、越南法定税务/发票系统
